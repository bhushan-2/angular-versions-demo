Earlier version of Angular i.e. Angular or Angular 1.x worked in project using Controllers now in Angular 2 its work around changed to Components. In real sense, Controllers still exist as one part of what we are calling a Component. The Component is composed and has a View and a Controller. The View is what we have HTML template while Controller in actual the JavaScript behaviour. 

ng generate component my-component

In Angular, Directives are defined as classes that can add new behavior to the elements in the template or modify existing behavior.The purpose of Directives in Angular is to maneuver the DOM, be it by adding new elements to DOM or removing elements and even changing the appearance of the DOM elements.

1. Component Directive
2. Structural Directive
3. Attribute directive
ng generate directive my-directive

Angular 2 has been designed with mobile in mind, it is designed to offer extensive support for mobile devices where touch events and gestures work across all devices.

1. It supports Single (Universal) Framework for both Web and Mobile device which reduces learning curve to most of the developers.
2. It has great mobile features like touch support, low memory, material design UI elements and performance tuning etc, which will help quickly resolve mobile related issues.
